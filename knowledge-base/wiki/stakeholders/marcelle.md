# Marcelle — Gestora do Projeto Principal

## Resumo

Marcelle é a gestora do projeto principal da Assinatura — a gênio operacional por trás do seu setor. Diferente da Gabi (que lida com expansão e novos negócios), Marcelle cuida da operação central da empresa. Sua necessidade é visibilidade: saber onde cada projeto está, sem precisar perguntar manualmente. O diagnóstico presencial com ela ainda está pendente.

## Detalhes

### Papel na Empresa

| Campo | Valor |
|---|---|
| Cargo | Gestora do Projeto Principal / Operações |
| Responsabilidade | Projeto central da Assinatura — gestão operacional |
| Perfil | "Gênio operacional" — foco em execução e controle de projetos |
| Principal dor | Falta de visibilidade do status dos projetos sem precisar perguntar |
| Ferramenta principal | Asana (gestão) + WhatsApp (comunicação) |

### Mapeamento de Necessidades (diagnóstico indireto — presencial pendente)

| Necessidade | Solução no Projeto |
|---|---|
| Saber status de cada projeto sem perguntar | Agente reativo no WhatsApp ("como está o projeto X?") |
| Visibilidade do ciclo de vida dos clientes | Controle de estágio com funil de projetos |
| Alertas de atrasos e erros | Reports automáticos via WhatsApp |
| Visibilidade interna e para o cliente | Dashboard com status compartilhável |
| Ver tudo em uma tela (desejo futuro) | **Fora do escopo atual** — exige UX dedicada |

### Status do Diagnóstico

- ✅ Diagnóstico **indireto** concluído — base levantada via análise do Asana
- ⏳ Diagnóstico **presencial** pendente — agenda a confirmar
- ⚠️ Escopo pode expandir após a sessão presencial

### Sequência no Projeto

```
Secretária A.I. (Gabi) — 4 meses → concluída → infra pronta
  ↓
Agente Marcelle — inicia usando a mesma infraestrutura (Evolution API + EasyPanel)
  → Sem custo adicional de infraestrutura
```

## Decisões Tomadas

- Agente Marcelle inicia **após** a Secretária A.I. — foco na qualidade de entrega de cada solução.
- Reutilização da infraestrutura já construída para Gabi — multiplicador de valor sem custo adicional.
- Dashboard unificado ("sonho" de Marcelle) **fora do escopo** atual.

## Learnings

- O diagnóstico indireto (via Asana) é uma fonte de incerteza — o escopo real pode ser maior do que o acordado.
- Realizar a sessão presencial o quanto antes para fechar o escopo com precisão.
