# Bot Marcelle

Bot de WhatsApp integrado com Asana, Google Drive e Gemini, focado na gestão operacional e acompanhamento de equipe/clientes.

## Funcionalidades
- **Integração Asana**: Criação e consulta de tarefas.
- **Google Drive**: Upload de arquivos e áudios.
- **Gemini**: Processamento de áudios e geração de resumos/atas.
- **Evolution API**: Comunicação via WhatsApp.

## Setup
1. Clone o repositório
2. Copie `.env.example` para `.env` e preencha as variáveis
3. Execute `docker-compose up -d`
